# API 키 발급·설정 가이드 (현실판)

> 청바지 MVP를 샘플 데이터에서 실데이터로 전환하기 위한 외부 API 발급 절차.
> **2026-05 기준 — 개인 개발자 입장에서 실제로 받을 수 있는지** 정직하게 표기.

---

## 🟢 즉시 가능 — 개인 가입만으로 OK

### ① 공공데이터포털 — 한국노인인력개발원 구인정보 API

- **URL**: https://www.data.go.kr/data/15015153/openapi.do
- **개인 발급**: ✅ 회원가입 후 활용신청 → 자동 승인 (당일~1일)
- **무료** · 일 1만건 제한
- **환경변수**:
  ```bash
  KORDI_API_KEY=발급받은_인증키
  ```
- **단점**: 일부 공고 목록만 개방 — 워크넷처럼 풍부하진 않음

### ② Kakao 로컬 API (Geocoding · 지도)

- **URL**: https://developers.kakao.com
- **개인 발급**: ✅ 카카오 계정으로 즉시 발급 (개인·사업자 무관)
- **무료** · 월 30만건
- **환경변수**:
  ```bash
  KAKAO_REST_API_KEY=발급받은_REST_키
  NEXT_PUBLIC_KAKAO_JS_KEY=프론트엔드용_JS_키
  ```
- **사용**: 주소→좌표, 거리 계산, 지도 임베드 (S2)

### ③ OpenAI API (AI 챗봇)

- **URL**: https://platform.openai.com
- **개인 발급**: ✅ 카드 등록 즉시
- **유료 (사용량 기반)** · MVP 월 예산 10만원 권장
- **환경변수**:
  ```bash
  OPENAI_API_KEY=sk-...
  ```

## ✅ 지금 당장 가능한 액션

1. **카카오 REST API 키** — 개인 OK · 즉시 발급 → 지도·거리 표시 진짜 데이터로
2. **공공데이터포털 노인일자리 구인정보** — 개인 OK · 1~2일 → 보조 일자리 데이터
3. **OpenAI 키** — 개인 OK · 즉시 → AI 챗봇 진짜 답변

이 3개만 받으셔도 시연 품질이 크게 올라갑니다. 사업자등록 전까지 워크넷 어댑터는 코드만 준비된 상태로 대기 — 등록 즉시 활성화 (코드 수정 X).

## 환경 변수 적용 방법

1. `mockup-app/.env.local` 파일 생성
2. 위 환경 변수들 입력
3. `npm run dev` 재시작 — 어댑터가 자동 활성화 (`isAvailable()` 체크)
4. 상태 확인: `curl http://localhost:3000/api/admin/ingestion`
5. Vercel: Settings → Environment Variables 에서 동일 키 추가

## 운영 자동화 (S2 이후)

- **Vercel Cron**: 1시간마다 `/api/admin/ingestion` POST 트리거
- **마감 정리**: 6시간마다 `filterExpired()` 자동
- **실패 알림**: Sentry · Slack Webhook (S2)
